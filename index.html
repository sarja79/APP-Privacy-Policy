Universal Code Viewer - Privacy Policy

Effective Date: 2026-03-03

1. Overview

Universal Code Viewer ("the App") is developed by Lee Jung Ouk (Package: kr.co.jia.highlighter).

Universal Code Viewer is a mobile application that allows users to open and view local code files and documents with syntax highlighting and markdown rendering. The app is available to all users and does not require login or account registration.

2. Information Collection

This app does not collect, store, or transmit any personal data to any server operated by the developer.

3. Advertising

This app uses Google AdMob to display banner advertisements.

Google AdMob may collect certain information automatically, including:

  • Advertising ID
  • Device information
  • IP address
  • App interaction data

This data is collected and processed by Google in accordance with their privacy policy:

https://policies.google.com/privacy

4. Data Sharing

The developer (Lee Jung Ouk) does not collect or share user data. However, third-party services such as Google AdMob may process data as described above.

5. User Rights

Users can reset or opt out of personalized ads by adjusting their device settings:

  • Android: Settings > Google > Ads

6. Children's Privacy

This app is not directed to children under the age of 13 and does not knowingly collect personal information from children.
